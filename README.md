# Natasha-AI

Natasha AI is a conversational AI assistant built with LangGraph, Groq, Streamlit, and ChromaDB.

## Features

- 💬 Conversational AI
- 📄 PDF document upload
- 🔍 Retrieval-Augmented Generation (RAG)
- 🧠 ChromaDB vector storage
- ⚡ Groq LLM integration
- 🌊 Streaming responses
- 🗂 Multiple chat sessions
- ✨ AI-generated chat titles
- 🔀 LangGraph workflow orchestration

## Tech Stack

- Python
- Streamlit
- LangGraph
- LangChain
- Groq API
- ChromaDB
- Sentence Transformers
